About

Built to keep lead collection practical

Metropolis is a lightweight demo brand for teams that want simple product pages, clear messaging, and CRM-connected forms without a heavy frontend build.

We focus on the basics that matter: pages that load quickly, forms that submit reliably, and analytics that are easy to configure. This project keeps the implementation static and straightforward so it is easy to host and extend.

The current version includes a dedicated forms page with both HubSpot and Creatio integrations, plus a shared Google Analytics snippet across the site.
