Implementation Notes

Shared header and footer blocks are injected with JavaScript so every page stays consistent.

UTM parameters are stored in session storage and automatically re-applied to internal navigation links.

Button clicks, navigation interactions, downloads, and outbound traffic are logged to the console for debugging.

The contact form sends a JSON payload that includes TrackingUserId captured from Creatio.

Longer sections like this give you scroll behavior to inspect alongside click events and page transitions.

Swap the placeholder GA4 measurement ID with your production value when you are ready.
